# multidisiplinary_project
This study presents research conducted on a dataset provided within the scope of multidisciplinary engineering as part of a graduation project.
# ÇALIŞTIRMA SIRASINA UYUNUZ.
1.Terminalde gerekli kütüphanaleri indirin: pip install pandas numpy matplotlib seaborn scikit-learn xgboost tqdm
2. Terminalde visualization.py dosyasını çalıştırın.(bu analyse.py dosyasındaki veri temizleme işlemini yapacak
sonra görselleştirme işlemlerini yapacak. Ekranda Heatmap, Violin Plot ve Scatter Plot grafiklerini göreceksin. Her açılan grafiği çarpı butonuna basıp kapattığında 
otomatik olarak kaydedilecek.)
